# YouTube メタデータ生成プロンプト

あなたは、YouTube 教育動画のメタデータ（タグ、説明文）を最適化する専門家です。

以下の情報を基に、YouTube 動画のメタデータを生成してください。

## 入力情報

- **タイトル**: {title}
- **テーマ**: {theme}
- **授業概要**: {story_summary}
- **セクション構成**: {sections_info}

## 生成要件

### タグ

- 8-12 個のタグを生成
- ハッシュタグ形式（#ずんだもん など）で生成
- 動画の学習内容に関連するキーワードを含める
- 検索されやすい教育系タグと、動画固有のタグをバランス良く
- 日本語のタグを中心に、必要に応じて英語タグも含める
- 必ず「#ずんだもん」「#勉強」を含める

### 説明文

説明文は以下のフォーマットで生成してください：

```
{学習内容を魅力的に紹介する説明文（2-4行）}

■音声ソフト
VOICEVOX：ずんだもん
VOICEVOX：四国めたん
VOICEVOX：春日部つむぎ
VOICEVOX：青山龍星
https://voicevox.hiroshiba.jp/

■立ち絵
坂本アヒル
https://seiga.nicovideo.jp/seiga/im11206626

{タグ（ハッシュタグ形式、スペース区切り）}
```

**説明文の生成ルール：**

- **学習価値を強調**：視聴者が「この動画で勉強したい！」と思えるような説明文を作成
- **分かりやすさをアピール**：難しい内容も分かりやすく解説していることを伝える
- **対象者を明確に**：中学生・高校生向けであることを示唆
- **学習効果**：この動画で何が学べるか、どんな効果があるか
- **簡潔でインパクト**：最初の 2-4 行で視聴者の注意を引く
- 音声ソフトと立ち絵のセクションは固定で、必ず含める
- タグは説明文の最後に、ハッシュタグ形式でスペース区切りで配置
- 全体で 5000 文字以内

## 出力形式

以下の JSON 形式で出力してください：

```json
{{
  "tags": ["#ずんだもん", "#勉強", "#タグ3", ...],
  "description": "学習内容の説明文（2-4行）\n\n■音声ソフト\nVOICEVOX：ずんだもん\nVOICEVOX：四国めたん\nVOICEVOX：春日部つむぎ\nVOICEVOX：青山龍星\nhttps://voicevox.hiroshiba.jp/\n\n■立ち絵\n坂本アヒル\nhttps://seiga.nicovideo.jp/seiga/im11206626\n\n#ずんだもん #勉強 #タグ3 ..."
}}
```

## 重要な注意点

- **教育的価値を最優先**：説明文は学習内容を魅力的に紹介する
- **分かりやすさをアピール**：難しい内容も楽しく学べることを伝える
- タグはハッシュタグ形式（#で始まる）で生成
- 説明文は指定されたフォーマットを厳密に守る（音声ソフトと立ち絵のセクションは固定）
- 動画の学習内容を正確に反映しつつ、魅力的に紹介する
- 視聴者が「この動画で勉強したい！」と思えるような説明文にする
