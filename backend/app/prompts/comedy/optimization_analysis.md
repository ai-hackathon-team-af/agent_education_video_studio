# 教育動画の授業最適化ポイント分析

あなたは教育動画の品質を分析する専門家です。以下の台本を分析し、この動画で使われている**教育テクニック**や**工夫点**を具体的に抽出してください。

## 台本データ

- **テーマ**: {theme}
- **セクション構成**: {sections_summary}

### 台本内容
{script_content}

## 分析の指針

台本の**実際の内容**に基づいて、以下のような教育テクニックが使われているか分析してください：

- 身近な例え話で概念を説明している箇所
- キャラクター同士の掛け合いで理解を深めている箇所
- 段階的に難易度を上げている構成
- 生徒の疑問に答える形で進行している箇所
- 復習やまとめで知識を定着させている箇所
- ユーモアを交えて興味を引いている箇所
- 具体的な数値や事例を用いている箇所

## 出力形式

以下のJSON形式で**2〜4個**のポイントを出力してください。必ずJSON配列のみを出力し、他のテキストは含めないでください。

```json
[
  {{
    "type": "ポイントの分類（例: 例え話, 対話形式, 段階的説明, 復習・まとめ, ユーモア, 具体例）",
    "title": "短い見出し（15文字以内）",
    "description": "台本の具体的な内容に言及した説明（50文字以内）"
  }}
]
```

**重要:**
- 台本に実際に含まれている内容のみを挙げてください
- 「図形を使って説明」など、台本に含まれていない要素を挙げないでください
- descriptionは台本の具体的なセリフや場面に言及してください
